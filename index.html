skatlogs

A real-time video feedback synthesizer in the browser.

Input Sources

  • Camera – Webcam feed
  • Screen/Tab – Screen or browser tab capture
  • Phosphorm – Built-in oscilloscope (choose audio source on click)
  • Test Pattern – Animated test signal

Pipeline

Signal flows: Input → [pipeline stages] → Delay Buffer → Screen

  • Spectral – Mesh displacement driven by brightness
  • Mixer – Core feedback: blends input with delayed frames, applies HSB transforms
  • Blur / Sharpen – Post-processing effects
  • Custom Shaders – User GLSL fragment shaders (uniforms: u_tex, u_resolution, u_time)

Reorder stages in the Pipeline Order section using arrow buttons.

Feedback Buffers (FB 0–3)

Up to 4 independent feedback layers. Each has blend, luma keying, HSB modulation, translation, rotation, zoom, and delay (1–29 frames).

Custom Shaders

Write GLSL fragment shaders. Available uniforms:

  • u_tex – input texture (sampler2D)
  • u_resolution – canvas size in pixels (vec2)
  • u_time – elapsed time in seconds (float)

Output: out vec4 fragColor. UV coords in vUv (0–1).

Version header (#version 300 es) is auto-prepended if missing.
